📄 Dataset Overview

Historical daily sales data used to train time-series forecasting models.

  • Date range: 3 years of daily data
  • Frequency: Daily (365 rows/year)
  • Target column: sales
  • Load with pd.read_csv('sales.csv', parse_dates=['date'])
  • Check missing values: df.isnull().sum()
  • View stats: df['sales'].describe()

🧹 Preprocessing

Clean missing values and cap outliers before feature extraction.

  • Reindex to daily frequency: asfreq('D')
  • Fill gaps: interpolate(method='time')
  • IQR outlier detection: Q1 − 1.5×IQR → Q3 + 1.5×IQR
  • Winsorise (clip) extreme spikes
  • Final missing count: 0

⚙ Feature Engineering (23 features)
